## Running

> You need to have the latest version of [Docker](https://www.docker.com/) installed to run this example

OS X & Linux:

```sh
./run-all.sh
```

Windows:
```sh
run-all.bat
```

Access [http://localhost:8180/](http://localhost:8180/)
